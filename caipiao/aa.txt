     <h3>
      参数说明
     </h3>
     <ul class="explain" id="explain">
      <li>
       <strong>
        和值：
       </strong>
       开奖号码相加之和，范围为3-18。示例：号码1 4 6和值为11。
      </li>
      <li>
       <strong>
        遗漏：
       </strong>
       自上期开出到本期间隔的期数。
      </li>
      <li>
       <strong>
        分隔线：
       </strong>
       每五期使用分隔线，使横向导航更加清晰。
      </li>
      <li>
       <strong>
        遗漏分层：
       </strong>
       将当前遗漏按照"1-5，6-11，12-无穷大"分别用黑色、蓝色、红色标识。
      </li>
      <li>
       <strong>
        出现总次数：
       </strong>
       统计周期内，某个号码的累计出现次数。
      </li>
      <li>
       <strong>
        平均遗漏值：
       </strong>
       统计期数内遗漏的平均值（计算公式：平均遗漏＝统计期内的总遗漏数/(出现次数+1)）。
      </li>
      <li>
       <strong>
        最大遗漏值：
       </strong>
       是指统计周期内所有遗漏值的最大值。
      </li>
      <li>
       <strong>
        最大连出值：
       </strong>
       是指统计周期内所有连出值的最大值。
       <span class="hide" id="hideExplain">
        收起
        <i>
        </i>
